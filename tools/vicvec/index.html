VicVec User Guide

Game vector authoring, not a full vector engine

Author rest art, then pose animation over it.

VicVec is built around fast game-dev vector packs for Duhrng. Use Edit Rest to create the source shapes. Use Animate to write sparse keys at the playhead without changing the rest art.

Mental model Interface Static pack Selection Cleanup Animation Export Troubleshooting

Core Mental Model

The rule: Edit Rest changes the actual vector art. Animate previews and keys changes over that art.

A pack contains shapes. Shapes contain loops. Loops contain points and relative Bezier handles. Roles and tags give the loops game meaning, such as outer art, cutouts, detail, hit zones, and effect zones.

  • Edit Rest: draw, clean, optimize, transform, style, tag, and organize the base art.
  • Animate: move loops, move points, edit handles, brush points, or change styles as keyed poses.
  • Auto Key: when on, canvas edits in Animate mode write sparse graph keys at the playhead.

Quick Start: Static Vector Pack

  1. Stay in Edit Rest.
  2. Optional: open a source image from the Pack panel.
  3. Use Point to place loop points. Click the first point again to close the loop.
  4. Use Loop to set the role and loop id.
  5. Use Object for shape id, label, tags, and z order.
  6. Use Style for fill, stroke, and stroke width.
  7. Use Export v1 for the compatibility-safe Duhrng pack.

Selection And Editing

  • Point: click points to select them. Shift-click toggles point selection. Empty canvas adds points.
  • Path Select: click a filled loop to select it. Shift-click toggles loops.
  • Lasso: select multiple points inside selected paths. Hold S while using Lasso to select paths under the lasso.
  • Move: selected points move first. If no points are selected, selected paths move.
  • Brush: default affects all points under the brush across shapes. Enable Selected Only to limit it.

Cleanup, Optimize, And Transform

Use cleanup for small repairs and Optimize Path for aggressive point reduction. Optimize always previews first.

  • Remove Near-Duplicates: removes consecutive points that are almost on top of each other.
  • Simplify Straight: removes middle points that barely change a straight segment.
  • Close Gap: snaps a tiny open gap and closes the loop.
  • Reverse: reverses point order while preserving curve continuity.
  • Transform Selection: preview scale, rotation, and origin choices before applying to rest geometry.

Animation Workflow

If you want motion, use Animate mode. If you want to redraw the source shape, use Edit Rest.
  1. Select the loop or points you want to animate.
  2. Open the Anim panel and add a clip.
  3. Click Add Track. This enters Animate mode and creates graph rows for the current target.
  4. At 0ms, click Set Key or Rest Key Selected.
  5. Move the playhead to a later time.
  6. Pose directly on the canvas with Move, Brush, point handles, or style controls.
  7. Click Set Key if the pose was edited through numeric fields, or to confirm the selected row.
  8. Scrub or press Play to preview the pose over the unchanged rest art.

Graph animation stores only changed outputs, such as loop transforms, point deltas, handle deltas, fill, stroke, stroke width, and opacity. It does not bake every frame.

Export Modes

Export v1
Compatibility-safe static Duhrng pack. This omits schema-v2 graph animation.
Save JSON
Saves the current compatibility-safe JSON unless the export drawer is in graph mode.
Export Graph v2
Explicit animation export for the sparse graph contract. Use this for runtime handoff.

Editor-only state such as selection, tool mode, previews, brush settings, playhead, FPS, and open drawers is not exported.

Troubleshooting

Animation changed my real points.
Switch to Animate before posing. Add Clip and Add Track now do this automatically.
Nothing plays.
Check that the timeline has graph rows, Preview is on, Auto Key is on, and keys exist at different times.
The start pose does not come back.
Add a rest key at 0ms or use Rest Key Selected before posing later frames.
Brush moved too much.
Lower Strength or Radius, or enable Selected Only before brushing.
Duhrng ignores graph animation.
Use Export v1 for static compatibility until the runtime graph evaluator is active.
